Buy Order Creation

To place a buy order there should be enough cash in the account to cover the estimated value of the security (based on the last trade) plus the trading fee.

Example

Given that an account holds $1000.00 in cash, CSCO is selling at $20.0000/share and the trading fee is $10.00/trade, when an order for 50 shares of CSCO is placed, the order will be rejected. However when an order for 49 shares of CSCO is placed, the order will be accepted,
