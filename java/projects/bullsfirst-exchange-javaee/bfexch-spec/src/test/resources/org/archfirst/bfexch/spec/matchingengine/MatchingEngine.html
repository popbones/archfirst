Matching Engine

The matching engine is responsible for accepting new orders and executing them. Buy orders are matched with sell orders. If a match occurs, both orders are executed. Details of the matching process are explained below.

  • Placing order in an empty book
  • Sorting orders on the buy stack
  • Sorting orders on the sell stack
  • Matching market orders
  • Matching limit orders
  • Matching buy market order with sell limit order
  • Matching buy limit order with sell market order
  • Matching AllOrNone orders
