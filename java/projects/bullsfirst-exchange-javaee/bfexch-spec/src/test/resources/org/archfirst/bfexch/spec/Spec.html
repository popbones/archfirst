Bullsfirst Exchange

Bullsfirst Exchange is a demonstration only trading exchange that shows best practices in software architecture such as Executable Specifications, Domain Driven Design and Test Driven Development.

This document is an executable specification of the Bullsfirst Exchange. Click the link below to explore the specification. As you navigate through different sections, you will find highlighted areas indicating success or failure of tests. Green highlights indicate successful tests, red highlights indicate failed tests.
