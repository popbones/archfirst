CSS3 Transitions

CSS3 Transitions allow property changes in CSS values to occur smoothly over a specified duration. The properties that can be animated are listed here.

By implication, CSS3 Transitions can only be used to change CSS property values. Don't confuse these transitions with page transitions that are common in modern mobile applications.

Instantaneous Property Change (without CSS Transition)

Property: background-color

Transition on hover

Property: background-color

Duration: 1s

Transition on activation (mouse click)

Property: background-color

Duration: 1s

Transition on hover

Property: width

Duration: 1s

Transition using JavaScript

Property: width

Duration: 1s
