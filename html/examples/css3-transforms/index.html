Notes

  • Click on each image to trigger the transform.
  • Effect desired is 3D rotation around Y axis.
  • CSS3 Transform provides the desired effect exactly.
  • jQuery Rotate can only rotate on x-axis.
  • jQuery Flip allows rotation on y-axis but the final image is same as the starting image. So this approach required switching to the rotated image before starting the animation.
